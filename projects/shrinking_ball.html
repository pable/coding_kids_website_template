Shrinking ball

  • Explanation
  • Code in Processing
  • Code in p5.js
  • Demo

Here we have a ball that follows your mouse

When you click the mouse, it'll grow until you release the button.

Then it'll shrink.

int ballSize = 0;
void setup(){
    size(900, 600);
}
void draw(){
    circle(mouseX, mouseY, ballSize);
    if(mousePressed){
        ballSize = ballSize + 1;
    }
    else{
        if(ballSize>0){
            ballSize = ballSize - 1;
        }
    }
}

let ballSize = 0;
